# Project Structures

This file shows how the project structure for each language hydra supports looks like.

### python

```
└── pythonDemo
   ├── .gitignore
   ├── LICENSE
   ├── README.md
   ├── pythonDemo
   |  └── __init__.py
   ├── setup.py
   └── tests
      ├── __init__.py
      └── test_pythonDemo.py
```


### go

```
└── goDemo
   ├── .gitignore
   ├── LICENSE
   ├── README.md
   ├── bin
   ├── go.mod
   ├── pkg
   ├── src
   |  └── main.go
   └── tests
      └── goDemo_test.go
```

### web

```
└── webDemo
   ├── .gitignore
   ├── LICENSE
   ├── README.md
   ├── css
   |  └── style.css
   ├── img
   ├── index.html
   └── js
      └── script.js
```

### flask

```
└── flaskDemo
   ├── .gitignore
   ├── LICENSE
   ├── README.md
   ├── app.py
   ├── static
   |  ├── images
   |  ├── scripts
   |  |  └── script.js
   |  └── styles
   |     └── style.css
   └── templates
      └── index.html
```


### c

```
└── cDemo
   ├── .gitignore
   ├── LICENSE
   ├── Makefile.am
   ├── README.md
   ├── libs
   ├── src
   |  ├── Makefile.am
   |  ├── main.c
   |  └── main.h
   └── tests
      ├── Makefile.am
      └── cDemo_test.c
```


### c++

```
└── cppDemo
   ├── .gitignore
   ├── CMakeLists.txt
   ├── LICENSE
   ├── README.md
   ├── include
   |  └── cppDemo
   |     └── header.h
   ├── libs
   ├── src
   |  ├── main.cpp
   |  └── main.h
   └── tests
      └── cppDemo_test.cpp
```

### ruby

```
└── rubyDemo
   ├── .gitignore
   ├── Gemfile
   ├── LICENSE
   ├── README.md
   ├── Rakefile
   ├── bin
   ├── lib
   |  └── rubyDemo.rb
   ├── rubyDemo.gemspec
   └── tests
      └── test_rubyDemo.rb
```
